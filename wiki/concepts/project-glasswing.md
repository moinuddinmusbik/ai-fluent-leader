# Project Glasswing

Anthropic's invite-only program for rolling out [[claude-mythos]] to vetted cybersecurity organizations before any public availability.

## Key facts (from Nate Herk's 2026-06-07 analysis)

- **Purpose:** Point Mythos at critical software to find and patch security holes. Reportedly uncovered serious bugs in old, widely-used code that had gone unnoticed for years.
- **Access model:** Strictly vetted partners — government entities and established cybersecurity companies only.
- **Scale:** ~50 partners at launch (April 2026) → 150 organizations across 15+ countries (June 2026).
- **Pricing:** $25/M input tokens, $125/M output.
- **Expansion rate:** 50 → 150 in ~6 weeks is the key leading indicator for when public readiness approaches.

## Why it matters as a leading indicator

Nate's heuristic: when Project Glasswing partner count crosses 500+, Anthropic's safety confidence is building and the timeline to some form of public access compresses. Track Anthropic's safety/policy pages monthly for partner count announcements.
