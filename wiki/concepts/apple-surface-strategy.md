# Apple Surface Strategy

The thesis, articulated by [[nate-b-jones]], that Apple's AI competitive position is built not on foundation model capability but on **owning the trusted device surface** where AI agents ultimately touch real user work.

## Core Claim

Whoever controls the OS layer through which AI agents interact with apps and users on a device collects rent from every model that runs there. Apple's 1B+ device install base, combined with App Intents (its OS-level agent integration API), positions it as the mandatory broker — not a model competitor.

## Evidence

- **App Intents framework** — Makes every third-party app callable by Apple's OS agent layer; developers must use Apple's API to be agent-accessible on Apple devices.
- **Google Gemini deal** — Interpreted by Nate not as defeat but as surface-first sequencing: borrow model capability now, own inference hardware later.
- **Nvidia partnership** — Points toward on-device inference, eliminating the cloud dependency and cementing Apple's hardware-software integration moat.
- **"First AI trillionaire" framing** — Nate's explicit scale claim for the surface-ownership play.

## Strategic Implication

AI value is moving toward whoever owns the trusted surface where agents touch real work. The model race is secondary; the surface race is primary.
