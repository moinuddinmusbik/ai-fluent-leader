# Claude Mythos

The model tier sitting above [[opus-4-8]] in Anthropic's model family. Described in Anthropic's own internal draft as "a step change — the most powerful model they've ever built."

## Key characteristics (from Nate Herk's 2026-06-07 analysis)

- **Cybersecurity specialty:** Extraordinarily capable at finding and patching security holes in old, widely-used code. The same capability can be used offensively, which is why access is strictly gated.
- **Project Glasswing:** Invite-only rollout to vetted cybersecurity organizations. Started at ~50 partners (April 2026), expanded to 150 organizations across 15+ countries (June 2026).
- **Pricing:** $25/M input tokens, $125/M output — 5x Opus 4.8 today.
- **Public availability:** Anthropic's April 2026 statement: "We do not plan to make Mythos preview generally available." Nothing has walked this back.

## Nate's three scenarios (as of 2026-06-07)

| Scenario | Description | Probability (Nate's lean) |
|----------|-------------|--------------------------|
| **Bull** | Limited, paid, gated Mythos later in 2026 — toned-down, GPT-5.6 timing pressure | Lower |
| **Base** | Capabilities fold silently into next Opus in H2 2026 or early 2027; Mythos name never reaches public | Highest |
| **Bear** | Stays locked to Glasswing partners forever; public gets watered-down Opus slice at best | Plausible |

## The "Mythos-class" distinction

Nate's pinned comment clarifies: Anthropic's "Mythos-class models in the coming weeks" language from the Opus 4.8 announcement refers to a **safeguarded version only** — "once stronger cyber safeguards are in place" — not the full-power Glasswing model.
