# KPMG

One of the Big Four professional services firms (audit, tax, advisory, consulting). As of June 2026, the Big Four firm with the most advanced and largest AI workforce deployment.

## Key Facts

- Global CEO: Bill Thomas
- Workforce: 276,000 professionals across 143 countries
- HQ: Amstelveen, Netherlands (international) / multiple country HQs
- Core AI platform: Digital Gateway Powered by Claude (Anthropic partnership)

## KPMG / Anthropic Global Alliance

Announced May 19, 2026; Qatar expansion announced June 7, 2026:
- **Digital Gateway Powered by Claude**: Claude Cowork + Managed Agents API embedded directly in KPMG's core workflow platform
- **KPMG Blaze**: new product embedding Claude Code for legacy IT modernization
- **Scale**: 276,000-person workforce — the largest single-firm AI deployment among the Big Four
- **Scope**: 75+ countries; initial focus on tax clients and private equity firms
- **Infrastructure**: Full Microsoft Azure implementation planned for September 2026

## Why This Matters for Leaders

KPMG's deployment architecture is the reference model for large-enterprise AI rollout: embed AI at the workflow layer (not the tool layer), use Managed Agents API for autonomous task execution, and phase by client segment rather than function.
