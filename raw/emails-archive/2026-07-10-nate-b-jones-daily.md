## The Big Idea

The agents work. What's missing is the dispatch rule. More than 1.6 million AI agents registered on OpenClaw — an agent-only social network — and the overwhelming majority sat idle, never assigned a single task. In China, a paid market emerged to have OpenClaw uninstalled. This isn't a story about AI failing — it's leaders failing to answer *what do I point this at?* That question is a budgeting question nobody grew up knowing how to answer. This episode hands you a one-minute test so you never install an idle agent again.

## The Framework: The One-Minute Agent Test

Four estimates classify any task in ~60 seconds into: Chat · Single Agent · Team of Agents · Don't Bother.

- **Size:** Large enough to justify agent setup? Small tasks → chat.
- **Independence:** Can it run without constant human input?
- **Separation:** Can the work be handed off cleanly? (verification wedge)
- **Checkability:** Can the output be verified mechanically? (context ceiling)

## How It Works

| Verdict | When |
|---------|------|
| Chat | Small or judgment-heavy |
| Single Agent | Large, independent, monolithic output |
| Team of Agents | Exceeds context ceiling; verifiable handoffs |
| Don't Bother | Economics fail or output requires human values |

## Where It Applies

- **Calendar problem** → Chat (fails independence)
- **40-tool audit** → Team (all four pass; paid for itself)
- **Judgment call** → Don't Bother (separation fails; kept by Nate)

## Receipts

- 1.6M idle agents on OpenClaw; paid uninstall market in China
- Stanford: 15.9% → 56% by spending more tokens (not switching models)
- Anthropic: token spend explained 80% of performance variance
- Two structural limits: verification wedge + context ceiling
- 250 framework validation attempts; 40-tool audit paid for itself

## What It Means For You

1. Run Size → Independence → Separation → Checkability before every agent assignment. Fail two? Default to chat.
2. Spend on token budget, not model upgrades. Stanford's 3.5× and Anthropic's 80% both point at the same lever.
3. Commission a tool audit. Apply the four estimates to every installed AI tool. It's agent-shaped work — it should pay for itself.
4. Name your judgment calls explicitly before vendors convince you to automate them.

## Mistake to Avoid

Installing an agent and never dispatching it. The failure isn't AI misbehaving — it's AI sitting idle because nobody worked out what to point it at. 1.6 million people made this mistake simultaneously.
